## 环境变量配置

```env
# ===== 数据库 =====
PGDATABASE_URL=postgresql://user:password@host:port/database

# ===== 认证 =====
JWT_SECRET=your-jwt-secret-key
ENCRYPTION_KEY=your-encryption-key

# ===== 管理员 =====
ADMIN_EMAIL=admin@example.com
ADMIN_PASSWORD=admin123
ADMIN_NAME=经理

# ===== SuperPay (支付宝) =====
SUPERPAY_BASE_URL=https://hixrs.ibpee.com:13758
SUPERPAY_MERCHANT_ON=your-merchant-on
SUPERPAY_MERCHANT_KEY=your-merchant-key

# ===== 九久支付 (微信) =====
JIUJIU_API_URL=http://bayq.hanyin.9jiupay.com
JIUJIU_MCH_ID=your-mch-id
JIUJIU_APP_SECRET=your-app-secret

# ===== 平台 (运行时自动注入) =====
COZE_PROJECT_DOMAIN_DEFAULT=https://xxx.dev.coze.site
COZE_WORKSPACE_PATH=/workspace/projects/
DEPLOY_RUN_PORT=5000
COZE_PROJECT_ENV=DEV|PROD
```

## 依赖管理

### 包管理器
- **pnpm**（强制）：仅允许使用 pnpm，禁止 npm/yarn

### 常用命令
```bash
# 安装依赖
pnpm install

# 添加依赖
pnpm add <package>
pnpm add -D <package>

# 移除依赖
pnpm remove <package>
```

## 运行时环境

### 开发环境
```bash
pnpm run dev     # 启动开发服务（端口 5000）
pnpm run build   # 构建前端
```

### 生产环境
```bash
pnpm run build   # 构建
pnpm run start   # 启动生产服务
```

### 端口约束
- **必须使用 5000 端口**
- 禁止使用 9000 端口（系统保留）

## 技术债务与限制

### 当前限制
1. 前端单文件构建，不支持代码分割
2. 后端使用 tsx 直接运行，生产环境建议预编译
3. 无 Redis 缓存层，高并发场景需扩展

### 待优化项
1. 添加 Redis 缓存支持
2. 实现后端 TypeScript 预编译
3. 添加 API 限流机制
