# 技术栈清单

本项目采用全栈 Node.js/TypeScript 架构，所有技术选型均基于 `package.json` 中的实际依赖。

## 1. 前端 (Frontend)
- **核心框架**: [React](https://react.dev/) (`v18.2.0`)
- **构建工具**: [Vite](https://vitejs.dev/) (`v5.2.0`)
- **路由管理**: 原生 React Context 状态管理（未引入 react-router，采用组件动态挂载单页应用架构）
- **UI & 样式**: [Tailwind CSS](https://tailwindcss.com/) (`v3.4.1`)
- **图标库**: [Lucide React](https://lucide.dev/) (`v0.344.0`)
- **动画库**: [Framer Motion / Motion](https://motion.dev/) (`v12.4.7`)

## 2. 后端 (Backend)
- **运行环境**: [Node.js](https://nodejs.org/) 
- **开发语言**: [TypeScript](https://www.typescriptlang.org/) (`v5.2.2`)，生产环境通过 `tsx` 执行 (`v4.7.1`)
- **Web 框架**: [Express](https://expressjs.com/) (`v4.19.2`)
- **数据库连接**: [pg](https://node-postgres.com/) (`v8.11.5`)
- **密码加密**: [bcryptjs](https://www.npmjs.com/package/bcryptjs) (`v2.4.3`)
- **身份认证**: [jsonwebtoken](https://jwt.io/) (`v9.0.2`)
- **图像处理**: 
  - [Canvas](https://www.npmjs.com/package/canvas) (`v2.11.2`) - 用于生成推广海报
  - [Sharp](https://sharp.pixelplumbing.com/) (`v0.33.2`) - 用于压缩和处理上传的商品图片
- **文件上传**: [multer](https://www.npmjs.com/package/multer) (`v1.4.5-lts.1`)
- **跨域处理**: [cors](https://www.npmjs.com/package/cors) (`v2.8.5`)
- **请求解析**: `body-parser` / `express.json`

## 3. 数据库 (Database)
- **核心引擎**: [PostgreSQL](https://www.postgresql.org/)
- **数据结构**: 
  - `users`: 用户表（含首席工程师、经理、主管、员工）
  - `products`: 商品表
  - `orders`: 订单表（含支付状态、分润金额）
  - `wallets`: 钱包表
  - `earnings`: 收益流水表
  - `withdrawals`: 提现申请表
  - `system_configs`: 系统全局配置表

## 4. 部署与运维 (DevOps)
- **进程管理**: [PM2](https://pm2.keymetrics.io/)
- **自动化部署**: GitHub Actions (`.github/workflows/deploy.yml`)
- **代码规范**: ESLint (`v8.57.0`)
- **Git 钩子**: Husky (`v9.0.11`)

---
*注：本项目不包含复杂的 ORM 框架（如 Prisma 或 TypeORM），所有数据库操作均通过 `pg` 模块手写原生 SQL 语句实现，以确保执行效率和首席工程师的直接可控性。*
