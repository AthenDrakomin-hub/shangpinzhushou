## 🗄️ 数据库表结构

### users 用户表

```sql
CREATE TABLE public.users (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  email VARCHAR(255) UNIQUE NOT NULL,
  encrypted_password VARCHAR(255) NOT NULL,
  display_name VARCHAR(100) NOT NULL,
  role VARCHAR(20) DEFAULT 'supervisor',  -- manager/supervisor/staff
  status VARCHAR(20) DEFAULT 'active',    -- active/inactive/pending
  merchant_on VARCHAR(50),                 -- 商户编号
  secret_key VARCHAR(100),                 -- 密钥
  whitelist_ip TEXT,                       -- IP白名单
  earnings_rate DECIMAL(5,4) DEFAULT 0.1, -- 佣金比例
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
  updated_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);
```

| 字段 | 类型 | 说明 | 默认值 |
|------|------|------|--------|
| id | UUID | 主键 | 自动生成 |
| email | VARCHAR(255) | 登录邮箱（唯一） | - |
| encrypted_password | VARCHAR(255) | 加密密码 | - |
| display_name | VARCHAR(100) | 显示名称 | - |
| role | VARCHAR(20) | 角色：manager/supervisor/staff | supervisor |
| status | VARCHAR(20) | 状态：active/inactive/pending | active |
| merchant_on | VARCHAR(50) | 商户编号 | 自动生成 |
| secret_key | VARCHAR(100) | API密钥 | 自动生成 |
| whitelist_ip | TEXT | IP白名单（JSON数组） | null |
| earnings_rate | DECIMAL(5,4) | 佣金比例 | 0.1 |

---

### products 商品表

```sql
CREATE TABLE public.products (
  id VARCHAR(50) PRIMARY KEY,
  user_id UUID REFERENCES public.users(id) ON DELETE CASCADE,
  name VARCHAR(255) NOT NULL,
  price DECIMAL(12,2) NOT NULL,
  original_price DECIMAL(12,2),
  description TEXT,
  image TEXT,
  sales INTEGER DEFAULT 0,
  stock INTEGER DEFAULT 0,
  template_id VARCHAR(50),
  status VARCHAR(20) DEFAULT 'active',
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
  updated_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);
```

| 字段 | 类型 | 说明 | 默认值 |
|------|------|------|--------|
| id | VARCHAR(50) | 商品ID（如P20260401ABC） | 自动生成 |
| user_id | UUID | 所属用户ID | 外键 |
| name | VARCHAR(255) | 商品名称 | - |
| price | DECIMAL(12,2) | 销售价格 | - |
| original_price | DECIMAL(12,2) | 原价 | null |
| description | TEXT | 商品描述 | null |
| image | TEXT | 商品图片URL | null |
| sales | INTEGER | 销量 | 0 |
| stock | INTEGER | 库存 | 0 |
| template_id | VARCHAR(50) | 海报模板ID | null |
| status | VARCHAR(20) | 状态：active/inactive | active |

---

### orders 订单表

```sql
CREATE TABLE public.orders (
  id VARCHAR(50) PRIMARY KEY,
  user_id UUID REFERENCES public.users(id) ON DELETE CASCADE,
  product_id VARCHAR(50) REFERENCES public.products(id),
  product_name VARCHAR(255),
  amount DECIMAL(12,2) NOT NULL,
  status VARCHAR(20) DEFAULT 'pending',
  buyer_name VARCHAR(100),
  buyer_phone VARCHAR(20),
  pay_type VARCHAR(50),
  pay_url TEXT,
  transaction_id VARCHAR(100),
  paid_at TIMESTAMP WITH TIME ZONE,
  expired_at TIMESTAMP WITH TIME ZONE,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
  updated_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);
```

| 字段 | 类型 | 说明 | 默认值 |
|------|------|------|--------|
| id | VARCHAR(50) | 订单ID | 自动生成 |
| user_id | UUID | 所属商户ID | 外键 |
| product_id | VARCHAR(50) | 商品ID | 外键 |
| product_name | VARCHAR(255) | 商品名称快照 | - |
| amount | DECIMAL(12,2) | 订单金额 | - |
| status | VARCHAR(20) | 状态 | pending |
| buyer_name | VARCHAR(100) | 买家姓名 | null |
| buyer_phone | VARCHAR(20) | 买家电话 | null |
| pay_type | VARCHAR(50) | 支付方式 | null |
| pay_url | TEXT | 支付链接 | null |
| transaction_id | VARCHAR(100) | 交易流水号 | null |
| paid_at | TIMESTAMP | 支付时间 | null |
| expired_at | TIMESTAMP | 过期时间 | null |

**订单状态流转**：
```
pending (待支付) → paid (已支付) → completed (已完成)
       ↓
   expired (已过期)
       ↓
   cancelled (已取消)
```

### 支付渠道对比

| 渠道 | 支付方式 | 回调支持 | 适用场景 |
|------|----------|----------|----------|
| **九久支付** | 微信扫码 | ✅ 自动回调 | 微信收款 |
| **SuperPay** | 支付宝扫码 | ✅ 自动回调 | 支付宝收款 |
| **银行卡转账** | 线下转账 | ❌ 手动确认 | 无第三方支付时 |

### 支付回调

```
回调地址：https://[您的域名]/api/orders/callback
```

回调处理：
1. 验证签名
2. 更新订单状态
3. 更新钱包余额
4. 发送通知（可选）

## ❓ 常见问题

### Q: 新员工如何获取账号？
A: 员工账号由管理员（经理/主管）统一创建，前端不再提供注册功能。

### Q: 微信内无法完成支付？
A: 请点击右上角「···」选择「在浏览器中打开」，然后重新下单。

### Q: 提现多久到账？
A: 审核通过后，通常 1-3 个工作日到账（取决于收款方式）。

### Q: 订单一直显示「待确认」？
A: 银行卡转账订单需要商户手动确认；在线支付订单如未自动确认，请联系客服。

### Q: 如何更换支付方式？
A: 支付方式由系统配置，如需更换请联系管理员。
