# 产品概述

> 文档版本：1.0.0 | 更新日期：2026-04-01

---

## 产品定位

**商品页助手 (payforme-helper)** 是一个面向个人商户、小微商家的商品展示页生成工具。

### 核心价值

帮助用户快速创建商品分享页面，实现：
- 📦 **商品展示**：一键生成精美商品展示页
- 💳 **多渠道收款**：支持支付宝、微信、银行卡转账
- 📊 **订单管理**：实时订单跟踪和状态管理
- 💰 **资金管理**：钱包余额、佣金分配、提现流程

---

## 目标用户

| 用户类型 | 典型场景 | 核心需求 |
|----------|----------|----------|
| **小微商户** | 个体经营者、地摊商家 | 快速创建商品页面、收款 |
| **代理服务商** | 代付、代充值业务 | 多渠道收款、订单管理 |
| **团队组织** | 有员工的商家 | 员工管理、佣金分配 |

---

## 核心功能模块

### 1. 用户管理
- **三级权限体系**：经理 > 主管 > 员工
- **账号创建机制**：员工账号由管理员统一创建，前端不提供注册入口
- **角色权限控制**：不同角色访问不同功能模块

### 2. 商品管理
- 商品创建、编辑、删除
- 商品图片上传与自动压缩
- 商品分享（链接/二维码/海报）
- 多模板海报生成

### 3. 订单系统
- H5 下单页面
- 多支付方式（微信扫码、支付宝扫码、银行卡转账）
- 订单状态流转（待支付 → 已支付 → 已完成）
- 订单统计与报表

### 4. 支付集成
- **SuperPay**：支付宝扫码收款
- **九久支付**：微信扫码收款
- **银行卡转账**：线下收款确认

### 5. 财务管理
- 员工钱包系统
- 佣金自动分配
- 提现申请与审核流程

---

## 用户角色与权限

| 角色 | 权限级别 | 可访问功能 |
|------|----------|------------|
| **经理 (manager)** | 最高权限 | 所有功能 + 用户审核 + 支付网关配置 + 创建主管/员工 |
| **主管 (supervisor)** | 商户管理 | 商品管理 + 订单管理 + 员工管理 + 提现审核 + 创建员工 |
| **员工 (staff)** | 基础权限 | 商品管理 + 订单查看 + 我的钱包 + 申请提现 |

## 产品特性

### 单文件部署
- 前端构建为单个 HTML 文件
- 便于部署和分发
- 无需复杂的服务器配置

### 微信环境适配
- 自动检测微信内置浏览器
- 引导用户在外部浏览器打开
- 优化微信环境下的用户体验

### 海报营销工具
- 8 种预设海报模板
- 支持自定义品牌标识
- 自动生成下单二维码

---

## 业务流程

### 商户使用流程
```
管理员创建账号 → 商户登录 → 创建商品 → 分享商品页 → 买家下单 → 确认订单 → 提现
```

### 买家下单流程
```
访问商品页 → 选择支付方式 → 填写信息 → 扫码支付 → 自动确认
```

---

## 技术约束

- **运行环境**：Coze 云端沙箱
- **端口限制**：必须使用 5000 端口
- **数据库**：PostgreSQL
- **认证方式**：JWT + bcryptjs
